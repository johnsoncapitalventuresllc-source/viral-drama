# Viral Drama

**Viral Drama** is a browser-based branching reality show story game set inside a high-tech reality TV house airing on the parody network **DramaPop TV**.

The player enters the house on day one and chooses how to survive: cause chaos, build alliances, float under the radar, or become the main character.

## Phase 1 Features

- Mobile-friendly title screen
- New game flow
- Player name entry
- Pronoun selection
- Fixed avatar selection
- Contestant archetype selection
- JSON-driven story scenes
- Dialogue box
- Choice buttons
- Stat tracking
- Relationship tracking
- Save/load with localStorage
- Reset save option
- Placeholder art system

## Local Testing

Open `index.html` in your browser.

## GitHub Pages

1. Upload all files and folders to the root of your GitHub repo.
2. Go to **Settings > Pages**.
3. Choose **Deploy from branch**.
4. Select `main` and `/root`.
5. Save.
6. Open your GitHub Pages URL.

## Notes

This project uses only HTML, CSS, JavaScript, and JSON.
No game engine is required.
